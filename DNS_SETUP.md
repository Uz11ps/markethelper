# Настройка DNS для домена iawuuw.com

## Проблема

Let's Encrypt не может выдать SSL сертификат, потому что домен `iawuuw.com` не указывает на ваш сервер.

## Решение

Нужно настроить DNS записи для домена `iawuuw.com`:

### 1. Найдите где управляется домен

Обычно это:
- Панель управления доменом у регистратора
- DNS хостинг (Cloudflare, Route53, и т.д.)

### 2. Добавьте A-запись

Добавьте следующие DNS записи:

```
Тип: A
Имя: @ (или iawuuw.com)
Значение: 80.76.43.75
TTL: 3600 (или Auto)

Тип: A
Имя: www
Значение: 80.76.43.75
TTL: 3600 (или Auto)
```

### 3. Проверка DNS

После добавления записей подождите 5-15 минут и проверьте:

```bash
# Проверка A записи
dig iawuuw.com +short
# Должно вернуть: 80.76.43.75

# Проверка www
dig www.iawuuw.com +short
# Должно вернуть: 80.76.43.75
```

### 4. Получение SSL сертификата

После того как DNS записи настроены:

```bash
cd /opt/markethelper
certbot --nginx -d iawuuw.com -d www.iawuuw.com
```

Certbot автоматически:
- Получит SSL сертификат
- Настроит Nginx для использования HTTPS
- Настроит редирект с HTTP на HTTPS

## Временное решение (без SSL)

Если DNS еще не настроен, можно временно использовать HTTP:

1. Убедитесь что Nginx использует временную конфигурацию:
   ```bash
   cp nginx.conf.temp /etc/nginx/sites-available/markethelper
   nginx -t
   systemctl reload nginx
   ```

2. Доступ к админ-панели будет по адресу:
   - http://80.76.43.75 (через IP)
   - http://iawuuw.com (после настройки DNS)

## После настройки DNS и SSL

Админ-панель будет доступна по адресу:
- **https://iawuuw.com**
- **https://www.iawuuw.com**
